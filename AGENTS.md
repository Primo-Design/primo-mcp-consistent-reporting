# Project Instructions

## Directory Layout

### `/html-reports`
- Directory to store all generated overview report html files. The filename shall be overview-{time period}.html. All overview report html files shall use /html-reports/styles.css. It is important that all report html files having the same structure, data points and design. It should be possible to easily convert the over report html files into Powerpoint and PDF files.

### `/company-logo.png`
- Shall be used in the header of any overview html report file.

### `/screenshot.png`
- Shows a reference image about design style guides.

### `/html-reports/styles.css`
- Initially this file does not exist. Before it gets created inspect screenshot.png and consult the user for any input.

- Never modify this file.
